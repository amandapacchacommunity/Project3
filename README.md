# Project3 — Museum Compensation Analysis

Project3 models compensation analytics for a large Manhattan museum with a $250M operating budget.

The application analyzes:

• 3-year workforce trends
• retention and turnover
• compensation distribution by department
• living wage comparisons (NYC benchmark)
• market survey benchmarking
• job ladder architecture (generic global framework)
• gala and development staffing analysis

The system generates synthetic workforce data and produces dashboard metrics similar to executive compensation reviews.
